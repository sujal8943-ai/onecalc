# 🚀 OneCalc - Premium All-In-One Calculator Hub

A beautiful, feature-rich multi-calculator web application with bilingual (English/Hindi) support. Built with modern HTML5, CSS3, and vanilla JavaScript.

## ✨ Features

### 9 Premium Calculators:
1. **🧮 Basic Calculator** - Standard arithmetic operations
2. **🎂 Age Calculator** - Calculate exact age from date of birth
3. **❤️ BMI Calculator** - Health & fitness assessment
4. **💰 EMI Calculator** - Monthly installment calculations
5. **📊 GST Calculator** - Tax calculations with multiple slabs
6. **📈 Percentage Calculator** - Quick percentage calculations
7. **💵 SIP Calculator** - Investment wealth estimator
8. **🏦 Loan Calculator** - Interest & total payable calculations
9. **🏷️ Discount Calculator** - Shopping offers & savings

### Additional Features:
- 🔍 Real-time calculator search functionality
- 🎨 Modern responsive design with premium styling
- 📱 Mobile-friendly interface
- 🌐 Bilingual support (Hindi/English)
- ⚡ Fast, instant calculations
- 🎯 Smooth animations and transitions
- 📺 Clean, intuitive user interface

## 📋 How to Use

### Locally:
1. Clone the repository:
```bash
git clone https://github.com/sujal8943-ai/onecalc.git
cd onecalc
```

2. Open `index.html` in your browser:
```bash
# On macOS
open index.html

# On Windows
start index.html

# On Linux
xdg-open index.html
```

### Online:
Simply visit: `https://sujal8943-ai.github.io/onecalc/`
